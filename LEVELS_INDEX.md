# FLN Levels Index

Quick reference for all 23 levels in the foundational numeracy learning progression. Each level directory lives under [`FLN Levels Structure/`](FLN%20Levels%20Structure/).

| Level | Concept Name | Brief Description |
|------:|---|---|
| 1 | [Quantity Comparison](FLN%20Levels%20Structure/Level%201/Level%201_%20Quantity%20%20Comparison.md) | Compare two groups of objects visually to identify equal, more, or less — no counting required. |
| 2 | [Odd One Out](FLN%20Levels%20Structure/Level%202/Level%202_%20Odd%20One%20Out.md) | Identify the object that does not belong to a group based on shape, size, colour, quantity, or category. |
| 3 | [Matching + Tracing Lines](FLN%20Levels%20Structure/Level%203/Level%203_%20Matching%20+%20Tracing%20Lines.md) | Match related objects, shapes, and quantities; trace lines to develop fine motor and association skills. |
| 4 | [Numbers 1–10](FLN%20Levels%20Structure/Level%204/Level%204_%20Numbers%201%E2%80%9310.md) | Recognise, trace, count, and match number names for numerals 1 through 10. |
| 5 | [Finger Gesture Counting](FLN%20Levels%20Structure/Level%205/Level%205_%20Finger%20Gesture%20Counting.md) | Connect finger representations with numerals and match groups of equal quantity. |
| 6 | [After, Between, Before](FLN%20Levels%20Structure/Level%206/Level%206_%20After,%20Between,%20Before.md) | Identify numbers that come before, after, and between given numbers in a sequence. |
| 7 | [Addition through Objects](FLN%20Levels%20Structure/Level%207/Level%207_%20Addition%20through%20objects.md) | Understand addition as combining two groups; solve problems within 10 using objects and numerals (no carrying). |
| 8 | [Subtraction (1–10)](FLN%20Levels%20Structure/Level%208/Level%208_%20Subtraction(1-10).md) | Understand subtraction as taking away objects; solve problems within 10 using objects and numerals (no borrowing). |
| 9 | [Pattern Recognition + Draw by Tracing](FLN%20Levels%20Structure/Level%209/Level%209_%20Pattern%20Recognition+Draw%20by%20Tracing.md) | Recognise, complete, and extend visual and numerical patterns; trace drawings for motor development. |
| 10 | [Comparison – Numeral](FLN%20Levels%20Structure/Level%2010/Level%2010_%20Comparison%20%E2%80%93%20Numeral.md) | Compare numbers using numerals to identify greater than, less than, and equal relationships. |
| 11 | [Review Assessment](FLN%20Levels%20Structure/Level%2011/Level%2011_%20Review%20Assessment.md) | Assess mastery of earlier concepts: comparison, matching, counting, patterns, and number sequencing. |
| 12 | [Tens and Ones](FLN%20Levels%20Structure/Level%2012/Level%2012_Tens%20and%20Ones.md) | Introduce place value; identify tens and ones in two-digit numbers using visual representations. |
| 13 | [Numbers 11–30](FLN%20Levels%20Structure/Level%2013/Level%2013_%20Numbers%2011%E2%80%9330.md) | Recognise, count, trace, and match number names for numerals 11 through 30. |
| 14 | [Counting + Fun Trace](FLN%20Levels%20Structure/Level%2014/Level%2014_%20Counting%20+%20Fun%20Trace.md) | Match object quantities with numerals; learn through creative tracing activities. |
| 15 | [After, Between & Before (1–30)](FLN%20Levels%20Structure/Level%2015/Level%2015_%20Mixed.md) | Apply before, after, and between concepts with harder sequences in the 1–30 number range. |
| 16 | [Addition (1–30)](FLN%20Levels%20Structure/Level%2016/Level%2016_%20Addition%20(1-30).md) | Solve addition problems involving numbers up to 30 using objects, pictures, and numerals (no carrying). |
| 17 | [Subtraction (1–30)](FLN%20Levels%20Structure/Level%2017/Level%2017_%20Subtraction%20(1-30).md) | Solve subtraction problems involving numbers up to 30 using objects and numerals (no borrowing). |
| 18 | [Ordering](FLN%20Levels%20Structure/Level%2018/Level%2018_%20Ordering.md) | Arrange numbers in ascending and descending order. |
| 19 | [Numbering 31–50](FLN%20Levels%20Structure/Level%2019/Level%2019_%20Numering%2031-50.md) | Identify, read, write, sequence, and match number names for numerals 31 through 50. |
| 20 | [Skip Counting in 2s/3s](FLN%20Levels%20Structure/Level%2020/Level%2020_%20Skip%20Counting%20in%202s_3s.md) | Identify and continue skip-counting patterns by 2s and 3s using number sequences and visual representations. |
| 21 | [Comparison (1–50)](FLN%20Levels%20Structure/Level%2021/Level%2021_%20Comparison%20(1-50).md) | Compare numbers up to 50 using >, <, and = symbols. |
| 22 | [Ordering (1–50)](FLN%20Levels%20Structure/Level%2022/Level%2022_%20Ordering.md) | Arrange numbers up to 50 in ascending and descending order. |
| 23 | [Review Assessment](FLN%20Levels%20Structure/Level%2023/Level%2023_%20%20Review%20Assessment.md) | Comprehensive review of concepts from Levels 12–22: number recognition, patterns, comparison, and ordering. |

## Sub-Level Structure

Each level may include sub-level files alongside the main level overview:

| File | Purpose |
|---|---|
| `N.0.md` | Main level — the mastery checkpoint students must clear to advance |
| `N.1.md` | Sub-level — same concept with additional scaffolding or easier framing |
| `N.2.md` | Sub-level — further remediation support for students who need it |

See [`minutes_of_meeting.md`](minutes_of_meeting.md) and individual level directories for sub-level details.
