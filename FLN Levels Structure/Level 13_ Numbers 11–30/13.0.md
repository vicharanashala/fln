# Main Level 13.0

### **Description**

This is the main assessment level for Level 13.0. Students demonstrate mastery of target concepts before progressing.

### **Sections**

#### **Section A: NUMBERS**

* Only the initial and end point will be shown
* All other places will be blank

#### **Section B: NUMBER NAMES**

* In this section the child needs to match or select the correct number name for each number.
