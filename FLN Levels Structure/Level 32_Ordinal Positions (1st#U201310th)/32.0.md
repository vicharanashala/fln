# Main Level 32.0

### **Description**

This level assesses mastery of ordinal positions. Students observe sequences of objects and write the correct position or identify the position of specific objects.

### **Sections**

#### **Section A: Write Position**

* Write the ordinal position of highlighted objects.

#### **Section B: Identify Position**

* Determine the position of an object in a sequence.

### **Examples**

🚗 🚕 🚌 🚙 🚓

(Highlight 🚌)

Answer: **3rd**

🐶 🐱 🐰 🦊 🐻

Position of 🦊 \= \_\_\_

Answer: **4th**
