# Sub-Level 32.1

### **Description**

This level covers the same concepts but students identify objects based on a given position. 

The focus is on locating rather than writing positions.

### **Sections**

#### **Section A: Circle the Given Position**

* Circle the object in the specified position.

#### **Section B: Match Position and Object**

* Connect ordinal numbers with objects.

#### **Section C: Find the Position**

* Locate objects in a sequence.

####
