# Main Level 35.0

### **Description**

This level evaluates students' understanding of all concepts learned from **Levels 24–34**. 

Students solve mixed questions that require applying multiple mathematical concepts independently. 

The assessment emphasizes conceptual understanding rather than memorization.

## **Sections**

### **Section A: Place Value & Number Sense**

* Tens and Ones  
* Numbers 51–100  
* Number Formation

### **Section B: Number Operations**

* Carry Addition  
* Borrow Subtraction  
* Multiplication through Repeated Addition

### **Section C: Number Relationships**

* Comparison  
* Ascending Order  
* Descending Order

### **Section D: Data & Time**

* Tally Marks  
* Reading Analog Clocks  
* Ordinal Positions

### **Section E: Measurement**

* Length  
* Weight  
* Capacity  
* Standard & Non-Standard Measurement
