# Main Level 1.0

### **Description**

This is the main level that every student has to clear before being promoted to the next level. This level introduces children to basic quantity comparison, focusing on identifying equal quantities and comparing more or less.

### **Sections**

#### **Section A: Equal Quantity**

* Match equal items across two columns.

#### **Section B: More/Less**

* Compare paired items or values and circle the one with the greater/lesser value.

### **Examples**

* Match the columns with equal objects (e.g., 3 balloons to 3 stars)
* Circle the group/number with the greater value (e.g., 5 vs 2)
