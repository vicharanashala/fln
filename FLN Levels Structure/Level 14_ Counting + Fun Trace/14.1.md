# Sub-Level 14.1

### **Description**

This sub-level provides structured practice and evaluation for concepts introduced in Level 14.

### **Sections**

#### **Section A: Matching (Object ↔ Number)**

* The child has to match the object with the correct numeral.
* The number range would be 1-20, but there must be a few questions from 21-30 as well

#### **Section B: Fun – Draw by Tracing**

* Number Range: 1-30
* This will contain more easy/obvious figures but numbering will be visible
