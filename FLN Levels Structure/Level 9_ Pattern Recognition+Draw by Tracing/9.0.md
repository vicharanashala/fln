# Main Level 9.0

### **Description**

This is the main assessment level for Level 9.0. Students demonstrate mastery of target concepts before progressing.

### **Sections**

#### **Section A: Pattern Recognition**

* The child needs to understand the pattern and draw the next.

#### **Section B: Draw by Tracing**

* This level involves creating a drawing by joining numbered dots, with 10 dots in one picture. It requires number recognition, counting, and following the correct sequence.
