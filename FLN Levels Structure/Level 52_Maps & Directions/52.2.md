

# **Sub-Level 52.2**

### **Description**

Students develop basic map-reading skills by recognizing directions, locating landmarks, and following simple one- or two-step routes.

## **Sections**

### **Section A: Identify Directions**

Example Activities

* Circle North.  
* Circle East.  
* Circle South.  
* Circle West.

(using arrows and a compass)

### **Section B: Locate Objects**

Example Activities

* Which object is north of the tree?  
* Which object is beside the house?

### **Section C: Follow a Simple Route**

Example Activities

* Start at the house. Move East. Where do you reach?  
* Start at the school. Move North. Where do you reach?

### **Section D: Picture Map**

Identify places using a simple map with 4–5 landmarks.
