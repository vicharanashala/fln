# Main Level 25.0

## **Description**

This level evaluates mastery of place value concepts. 

Students independently identify tens and ones, represent numbers in place value charts, compose and decompose two-digit numbers, and solve mixed place value questions using numbers up to 100\.

## **Sections**

### **Section A: Identify Tens and Ones**

* Write the tens and ones for the given number.  
* Identify the value of each digit.

**Examples**

67

Tens \= \_\_\_

Ones \= \_\_\_

84

Tens \= \_\_\_

Ones \= \_\_\_

### **Section B: Complete the Place Value Chart**

* Fill the missing number or place value.

**Examples**

| Tens | Ones | Number |
| ----- | ----- | ----- |
| 4 | 9 | \_\_\_ |
| \_\_\_ | 6 | 56 |
| 8 | \_\_\_ | 82 |

### **Section C: Expanded Form**

* Express numbers as tens and ones.

**Examples**

43 \= 40 \+ \_\_\_

68 \= \_\_\_ \+ 8
