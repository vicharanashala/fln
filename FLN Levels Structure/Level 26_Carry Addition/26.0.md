# Main Level 26.0

### **Description**

This level evaluates mastery of carry addition using two-digit numbers. 

Students independently solve mixed addition problems requiring regrouping while applying place value understanding. 

All answers will be less than 100\.

#### **Vertical Carry Addition**

* Add two-digit numbers requiring carrying.  
* Write carries correctly in the tens place.
