# Sub-Level 4.1

### **Description**

This sub-level provides structured practice and evaluation for concepts introduced in Level 4.

### **Sections**

#### **Section A: NUMBERS**

* Object counting with Number tracing.
* Increase the number of boxes .
* Independent practice.
* Add a hand gesture column with objects.

#### **Section B: NUMBER NAMES**

* In this section the child needs to trace the number name for each number.
