# Sub-Level 2.2

### **Description**

This sub-level provides the easiest introduction to the "odd one out" concept. It features highly distinct visual differences (e.g., three identical objects and one completely different object) to build baseline observation skills.

Identify the object that looks completely different in a small group of 3 items.

### **Examples**

* Identify the odd one out: [Ball, Ball, Cup] (The odd one out is Cup, as the other two are identical balls).
* Identify the odd one out: [Apple, Apple, Star] (The odd one out is Star).
