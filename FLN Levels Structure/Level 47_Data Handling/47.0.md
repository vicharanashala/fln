# **Main Level 47.0** 

### **Description**

Students independently interpret tally charts, pictographs, and bar graphs to answer questions, compare data, and identify trends.

## **Sections**

### **Section A: Tally Chart**

Read the tally chart and answer the questions.

| Fruit | Tally |
| ----- | ----- |
| 🍎 |  |
| 🍌 |  |
| 🍇 |  |
| 🍊 |  |

**Examples**

* Which fruit has the highest count?  
* Which fruit has the lowest count?  
* How many apples are there?

### **Section B: Pictograph**

**Key:** 🍎 \= 2 children

| Fruit | Picture |
| ----- | ----- |
| Apple | 🍎🍎🍎 |
| Mango | 🍎🍎 |
| Orange | 🍎🍎🍎🍎 |

**Examples**

* How many children like oranges?  
* Which fruit is liked the most?  
* How many more children like oranges than mangoes?

### **Section C: Bar Graph**

(Simple vertical bar graph with 4 categories.)

**Examples**

* Which bar is the tallest?  
* Which category has the least value?  
* How many students chose blue?

### **Section D: Mixed Interpretation**

* Compare two categories.  
* Find the difference.  
* Identify equal values.
