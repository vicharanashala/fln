# Main Level: 12.0

### **Description**

* This level contains standard place-value questions where students identify tens and ones in numbers from 11 to 30\.   
* Activities include counting bundles, matching representations to numbers, and determining the number of tens and ones in a given numeral.

### **Assessment Criteria**

* Correctly identifies tens and ones.  
* Matches representations with numerals.  
* Represents numbers using tens and ones.
