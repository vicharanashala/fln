# Main Level 29.0

### **Description**

This level evaluates mastery of ordering concepts using numbers up to 100\. Students independently arrange multiple numbers in ascending and descending order while applying place value understanding and number comparison skills.

### **Sections**

#### **Section A: Ascending Order**

* Arrange 4–5 numbers from smallest to largest.

#### **Section B: Descending Order**

* Arrange 4–5 numbers from largest to smallest.

#### **Section C: Place Value Ordering**

* Order numbers with close values using tens and ones.

### **Examples**

* Arrange in ascending order: 67, 45, 89, 52  
* Arrange in descending order: 74, 91, 63, 58  
* Order: 71, 17, 70, 27, 72
